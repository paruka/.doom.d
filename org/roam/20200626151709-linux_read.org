#+TITLE: Linux read function
#+ROAM_TAGS: linux_io linux

* 函数声明：
#+BEGIN_SRC C
#include <fcntl.h>

ssize_t read(int fd, void *buf, size_t nbytes);
#+END_SRC

返回值：
|--------------+--------|
| 结果         | 返回值 |
|--------------+--------|
| 失败         |     -1 |
| 读取的字节数 |     >0 |
| 读到文件结尾 |      0 |
|--------------+--------|
