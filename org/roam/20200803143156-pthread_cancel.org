* 描述 
线程通过调用此函数，取消同一进程中的其它线程。

在默认情况下，该函数使得由 **tid** 标识的线程的行为表现为，同调用了参数为 **PTHREAD_CANCELEN** 的[[file:20200803140155-pthread_exit.org][pthread_exit]]一致。

注意，该函数 **并不等待线程终止，它仅仅提出请求** 。

* 声明
#+BEGIN_SRC C
#include <pthread.h>

int pthread_cancel(pthread_t tid);
#+END_SRC
** 返回值
- 成功：0
- 失败：错误码
* 注意
该函数 **并不等待线程终止，它仅仅提出请求** 。

相应此请求时，会触发[[file:20200803145746-thread_cleanup_handler.org][thread cleanup handler]]的调用。
