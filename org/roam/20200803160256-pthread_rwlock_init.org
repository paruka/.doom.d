* 描述 
该系列函数负责对读写锁执行初始化、销毁操作。

* 声明
#+BEGIN_SRC C
#include <pthread.h>

int pthread_rwlock_init(pthread_rwlock_t *rwlock, const pthread_rwlockattr_t *attr);

int pthread_rwlock_destroy(pthread_rwlock_t *rwlock);
#+END_SRC
** attr
[[file:20200803160529-pthread_rwlockattr_t.org][pthread_rwlockattr_t]]是读写锁属性，如果希望创建默认属性的读写锁，可将 **attr** 属性设为 **NULL** 。
** 返回值
- 成功：0
- 失败：错误码
